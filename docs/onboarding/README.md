# Developer Onboarding

## Prerequisites

- Docker Desktop
- Go 1.22+
- Node.js 20 LTS (use `nvm use` from repo root)
- Python 3.8+
- `make`

## Quick Start

1. Clone the repository
2. Run `make docker-up` to start the local Fabric network
3. Run `make generate-data` to populate test data
4. Run `make api-dev` to start the REST API
5. See [local-dev-setup.md](local-dev-setup.md) for detailed instructions
